  • BigHPC

    Development of a monitoring framework for High Performance Computing.

    #Python #Grafana #Docker #Promscale #HPC

    BigHPC

    The monitoring framework was composed of a multi-stage approach to gather and analyze data from compute nodes.

    The first stage involved deploying monitoring python probe on individual compute nodes. This agent continuously collected a comprehensive range of metrics.

    The raw data collected from this agent then underwent processing to become more usable for analysis. This involved filtering out irrelevant information, converting data to consistent formats, and combining individual data points into meaningful summaries.

    Finally, the stored data was presented to users through dashboards. These visualization tools enabled users to not only monitor resource utilization in real-time but also to analyze trends and identify potential bottlenecks or performance issues over time.
